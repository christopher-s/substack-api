# Substack Post Editor - Complete Formatting & Widget Reference

> Generated by browser automation exploration of the Substack draft editor on 2026-04-29.
> Editor backend: TipTap (ProseMirror-based), using Mousetrap for keyboard handling.

---

## 1. Toolbar Layout

The editor toolbar (`role="toolbar"`) contains these buttons from left to right:

| # | Button | Type | Dropdown? | Description |
|---|--------|------|-----------|-------------|
| 1 | Undo | Icon | No | Undo last action |
| 2 | Redo | Icon | No | Redo last undone action |
| 3 | Style | Text | Yes | Text/heading style selector |
| 4 | Bold | Icon | No | Toggle bold |
| 5 | Italic | Icon | No | Toggle italic |
| 6 | Strikethrough | Icon | No | Toggle strikethrough |
| 7 | Code | Icon | No | Toggle inline code |
| 8 | Link | Icon | No | Insert/edit hyperlink |
| 9 | Image | Icon | Yes | Image insertion options |
| 10 | Headphones | Icon | Yes | Audio options |
| 11 | Video | Icon | No* | Opens video upload dialog |
| 12 | Quote | Icon | Yes | Quote/block types |
| 13 | Bullet List | Icon | No | Toggle bullet list |
| 14 | Ordered List | Icon | No | Toggle numbered list |
| 15 | Alignment | Icon | Yes | Text alignment (lucide: align-left) |
| 16 | Button | Icon | Yes | CTA/widget buttons |
| 17 | Template | Icon | Yes | Template management |
| 18 | More | Icon | Yes | Additional block types |

*Video opens a direct upload/embed dialog (not a dropdown menu).

Additional editor controls outside the toolbar:
- **Plus** button: Inline block insertion trigger
- **Email header / footer** button: Opens email customization settings page
- **Title** textbox: Post title
- **Subtitle** textbox: Post subtitle
- **Cover image**: Upload area for cover image

## 8. Keyboard Shortcuts Summary

### Markdown-style Input Shortcuts (type + space)

| Input | Result |
|-------|--------|
| `#` + Space | Heading 1 |
| `##` + Space | Heading 2 |
| `###` + Space | Heading 3 |
| `####` + Space | Heading 4 |
| `#####` + Space | Heading 5 |
| `######` + Space | Heading 6 |
| `>` + Space | Block quote |
| `*` + Space | Bullet list |
| `-` + Space | Bullet list |
| `1.` + Space | Ordered list |
| `---` | Horizontal rule (divider) |

### Selection-based Shortcuts

| Shortcut | Action |
|----------|--------|
| Cmd/Ctrl + B | Bold |
| Cmd/Ctrl + I | Italic |
| Cmd/Ctrl + Shift + X | Strikethrough |
| Cmd + K | Insert/edit link |
| Cmd/Ctrl + Z | Undo |
| Cmd/Ctrl + Shift + Z | Redo |
| Enter | New paragraph |
| Shift + Enter | Line break (within paragraph) |
